The PHP_browseSOS set of modules are designed to assist developers parse instances of the IOOS 52N SOS.

Installation

  1. Download the compressed ZIP file and extract the content onto a folder on the server that can be accessed via HTTP/HTTPS; and
  2. Edit the ogc_sos.config file to provide the proper SOS endpoint to use. Note that support for multiple SOS endpoint are already listed in the issue tracking.

Modules

The following are the preliminary list of modules developed.

                                                         $observationBound     geographic bounding

                                                                               value)
